# Dahan's Ice Cream PWA

מוכן ל-GitHub Pages.

## איך מעלים
1. צור ריפו חדש ב-GitHub.
2. העלה אליו את כל הקבצים שבתיקייה הזו.
3. ב-GitHub: Settings -> Pages
4. תחת Build and deployment בחר:
   - Source: Deploy from a branch
   - Branch: main / root
5. שמור.
6. אחרי דקה-שתיים תקבל כתובת אתר.

## איך מתקינים באייפון
1. פתח את הכתובת ב-Safari
2. לחץ על Share
3. בחר Add to Home Screen

## מבנה
- index.html
- manifest.json
- sw.js
- assets/

## הערה
אם תשנה קבצים ותעלה מחדש, ייתכן שתצטרך לרענן פעם אחת כדי שה-Service Worker יתעדכן.
